## 麻城市正欣会计培训中心考试管理系统
	该系统主要用于老师创建试卷提供学员考试

### 技术点
* spring-boot v1.5.8
* mongodb v3.4.3
* spring-security v4.2.3

### 业务
* 组织架构管理
* 考试管理
	* 题目管理
	* 班级管理
	* 试卷管理
* 学员考试
