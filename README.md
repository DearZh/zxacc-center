## 麻城市正欣会计培训中心考试管理系统
	该系统主要用于老师创建试卷提供学员考试

### 技术点
* spring-boot v1.5.8
* mongodb v3.4.3
* spring-security

### 业务
1. 组织架构管理
2. 考试管理
  1. 题目管理
  2. 班级管理
  3. 试卷管理
3. 学员考试
